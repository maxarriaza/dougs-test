# Dougs test

## Requirements

NodeJS version : 12.18
NPM version : 6.14.6

## Installation

```bash
$ npm install
```

## Quick start

Start the application by running following command in terminal: 
```bash
$ npm run start
```

## Test

Start unit by running following command in terminal: 
```bash
$ npm run test
```

To enable coverage report :
```bash
$ npm run test:cov
```
